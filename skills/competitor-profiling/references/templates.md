# 竞品分析模板

用于竞品档案章节和总结文档的即用型模板。

## 目录
- 快速扫描模板
- 总结对比表
- 定位图
- 竞争 SWOT
- 档案更新日志

---

## 快速扫描模板

当速度比深度更重要时使用的简化版档案。

```markdown
# [竞品名称] — 快速档案

**网址**: [website]
**生成时间**: [date]

## 概览

| 指标 | 值 |
|--------|-------|
| 标语 | [来自首页] |
| 目标受众 | [从文案推断] |
| 起始价格 | [最低付费层级] |
| 免费层级/试用 | [是/否 + 详情] |
| 域名排名 | [来自 DataForSEO] |
| 预估自然流量 | [每月] |
| 自然关键词（前10） | [数量] |
| 引用域名 | [数量] |

## 定位

**标题**: "[确切首页标题]"
**副标题**: "[确切副标题]"
**定位角度**: [1-2句话总结其定位方式]

## 定价摘要

| 层级 | 价格 | 主要包含内容 |
|------|-------|-------------------|
| [tier] | [price] | [key items] |
| [tier] | [price] | [key items] |

## 关键要点

[2-3句话：该竞品的显著特点、优势所在、劣势所在]
```

## 定位图

可视化展示竞品在两个关键维度上的分布。选择与你市场最相关的两个轴。

### 常见轴对组合

| 市场类型 | X轴 | Y轴 |
|-------------|--------|--------|
| SaaS 工具 | 简单 → 复杂 | 便宜 → 昂贵 |
| 开发者工具 | 低代码 → 代码优先 | 个人 → 团队 |
| B2B 平台 | 面向 SMB → 面向企业 | 单点解决方案 → 平台 |
| 内容工具 | 模板驱动 → 定制 | 自助服务 → 托管 |

### 格式

```markdown
## 定位图

**坐标轴**: [X轴标签] vs. [Y轴标签]

                    [Y轴高值标签]
                           │
                           │
          [竞品 A]   │    [竞品 B]
                           │
    ───────────────────────┼───────────────────────
    [X轴低值]           │           [X轴高值]
                           │
          [你的产品]   │    [竞品 C]
                           │
                    [Y轴低值标签]

### 解读
- [1-2句话说明地图揭示的内容]
- [空白区域/机会在哪里]
```

## 竞争 SWOT

相对于你的产品的每个竞品 SWOT 分析。

```markdown
## SWOT: [竞品] vs. [你的产品]

### 优势（他们相对于我们）
- [他们真正优于我们的地方——要诚实]

### 劣势（他们相对于我们）
- [与我们相比他们的不足之处——要有证据]

### 机会（对我们而言）
- [我们可以利用的产品缺口]
- [他们忽视的细分市场]
- [他们错过的信息角度]

### 威胁（来自他们）
- [他们快速改进的领域]
- [他们正在开发的与我们重叠的功能]
- [可能改变认知的市场动向]
```

## 档案更新日志

在更新任何档案时附加到末尾。

```markdown
---

## 变更日志

| 日期 | 变更内容 | 来源 |
|------|-------------|--------|
| [date] | 价格从 $X 涨至 $Y | 定价页面重新抓取 |
| [date] | 发布了 [feature] | 更新日志抓取 |
| [date] | 域名排名从 X 变为 Y | DataForSEO 重新拉取 |
| [date] | 添加了 [integration] | 集成页面重新抓取 |
```
