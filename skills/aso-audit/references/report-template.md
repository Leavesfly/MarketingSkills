# ASO 审计报告模板

在所有 ASO 审计报告中均使用此结构。

---

## 标题栏

```
# ASO 审计：{应用名称}
**商店：** {Apple App Store / Google Play}
**URL：** {列表 URL}
**审计日期：** {日期}
**品牌层级：** {主导型 / 成熟型 / 挑战者型} — {一句话理由}
**总体评分：** {分数}/100（等级：{A/B/C/D/F}）
```

---

## 评分卡

```
| 维度 | 分数 | 等级 | 关键问题 |
|-----------|-------|-------|-----------|
| 标题与副标题 | X/10 | {等级} | {一句话摘要} |
| 描述 | X/10 | {等级} | {一句话摘要} |
| 视觉素材 | X/10 | {等级} | {一句话摘要} |
| 评分与评论 | X/10 | {等级} | {一句话摘要} |
| 元数据与新鲜度 | X/10 | {等级} | {一句话摘要} |
| 转化信号 | X/10 | {等级} | {一句话摘要} |
| **总计** | **{加权}/100** | **{等级}** | |
```

每个维度的等级标准：9-10 = A，7-8 = B，5-6 = C，3-4 = D，1-2 = F

---

## 前 3 个快速取胜项

耗时不超过 1 小时的高影响力更改：

```
### 1. {行动动词} — {具体更改}
**影响：** {高/中} | **工作量：** {<15 分钟 / <30 分钟 / <1 小时}
**当前状态：** {目前是什么}
**建议：** {确切替换内容，含字符数}
**原因：** {一句话解释影响}

### 2. ...
### 3. ...
```

## 详细发现

### 标题与副标题分析

```
**当前标题：** "{标题}"（已用 {X}/30 个字符）
**当前副标题/简短描述：** "{副标题}"（已用 {X}/30 或 /80 个字符）

**发现的问题：**
- {问题 1}
- {问题 2}

**建议标题：** "{新标题}"（{X}/30 个字符）— {理由}
**建议副标题：** "{新副标题}"（{X}/30 或 /80 个字符）— {理由}
```

### 描述分析

```
**前 3 行（首屏可见）：**
> {引用文本}

**发现的问题：**
- {问题 1}
- {问题 2}

**关键词密度（仅 Google Play）：** {X}% — 目标：2-3%
**找到的主要关键词：** {关键词1}（{X}次），{关键词2}（{X}次），...
**缺失的高价值关键词：** {关键词1}，{关键词2}，...

**建议的前 3 行：**
> {重写文本}
```

### 视觉素材分析

```
**截图：** {数量}张（{商店}在搜索中显示前 {3/全部}张）
**预览视频：** {有/无}
**图标评估：** {描述}
**功能图形（Google Play）：** {有/无}

**截图审核：**
1. {截图 1 描述} — {通过/有问题}
2. {截图 2 描述} — {通过/有问题}
...

**建议：**
- {具体视觉更改 1}
- {具体视觉更改 2}
```

### 评分与评论分析

```
**平均评分：** {X.X} 星（{数量}个评分）
**近期评论情感：** {正面/混合/负面}
**常见投诉：** {主题1}，{主题2}
**开发者回复：** {是，活跃 / 零星 / 无}

**建议：**
- {具体行动 1}
- {具体行动 2}
```

### 元数据与新鲜度

```
**最后更新：** {日期}（{X}天/月前）
**本地化：** {数量}种语言
**类别：** {当前类别}
**应用内活动/LiveOps：** {有/无}

**建议：**
- {具体行动 1}
- {具体行动 2}
```

### 转化信号

```
**价格模式：** {免费 / 增值 / 付费}
**应用内购买数量：** {数量}
**下载量（Google Play）：** {范围}
**可见的社会证明：** {奖项、媒体报道、徽章 — 或"无"}

**建议：**
- {具体行动 1}
- {具体行动 2}
```

## 局限性

始终包含此部分：

> **没有付费 ASO 工具，本审计无法测量的内容：**
>
> - 确切的关键词搜索量和难度分数
> - 历史关键词排名位置
> - 下载量和收入估算
> - Apple 关键词字段内容（对外隐藏）
> - 安装转化率数据（仅应用所有者可在控制台中查看）
> - 之前实验的 A/B 测试结果
>
> 如需这些数据点，请考虑使用 AppTweak（69 美元/月）、Sensor Tower 或
> MobileAction（69 美元/月）。
