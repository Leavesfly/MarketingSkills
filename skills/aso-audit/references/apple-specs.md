# Apple App Store — 官方规范与指南

所有数据来自 developer.apple.com，截至 2026 年 3 月。

## 字符限制

| 字段                   | 限制            | 是否用于搜索索引？      | 说明                                                    |
| ----------------------- | ---------------- | ------------------------ | -------------------------------------------------------- |
| App 名称                | 30 个字符（最少 2 个） | 是                      | 必须唯一；不得使用商标、竞争对手名称、定价 |
| 副标题                | 30 个字符         | 是                      | 不得有无法验证的声明                                   |
| 关键词                | 100 字节        | 是（隐藏）             | 逗号分隔，术语之间无空格                          |
| 描述             | 4,000 个字符      | **否**                   | 仅纯文本，无 HTML                                 |
| 推广文本        | 170 个字符        | **否**（Apple 确认） | 无需新版本即可更新                            |
| 更新内容              | 4,000 个字符      | 否                       | 首个版本后的所有版本必需                    |
| IAP 名称                | 35 个字符         | 是                      | 出现在搜索结果中                                        |
| IAP 描述         | 55 个字符         | 否                       |                                                          |
| 应用内活动名称       | 30 个字符         | 是                      | 需要标题大小写                                      |
| 应用内活动简短描述 | 50 个字符         | 是                      | 句子大小写                                            |
| 应用内活动详细描述  | 120 个字符        | 否                       | 句子大小写                                            |

**关键词字段为 100 字节，而非 100 个字符。** 非拉丁脚本（阿拉伯语、中文、日语、韩语）每个字符使用 2-3 字节，显著减少有效关键词数量。

## 截图规范

| 设备           | 是否必需？     | 数量 | 尺寸（竖屏）      |
| ---------------- | ------------- | ----- | -------------------------- |
| 6.9" iPhone      | **必需**  | 1-10  | 1260 x 2736                |
| 13" iPad         | **必需**  | 1-10  | 2064 x 2752                |
| Mac              | 如适用 | 1-10  | 最多 2880 x 1800（16:10）  |
| Apple Watch      | 如适用 | 1-10  | 因型号而异            |
| Apple TV         | 如适用 | 1-10  | 1920 x 1080 或 3840 x 2160 |
| Apple Vision Pro | 如适用 | 1-10  | 3840 x 2160                |

- 格式：JPEG、PNG
- Apple 从必需的基础尺寸自动缩放到较小设备

## App 预览视频规范

- **数量：** 每个应用最多 3 个
- **时长：** 15-30 秒
- **最大文件大小：** 500 MB
- **编解码器：** H.264（10-12 Mbps，最高 30fps）或 ProRes 422 HQ
- **音频：** 立体声，256 kbps AAC 或 PCM，44.1/48 kHz
- **格式：** .mov、.m4v、.mp4
- **行为：** 在产品页面上静音自动播放（iOS 11+）

## 自定义产品页面 (CPPs)

- **最大数量：** 70 个附加页面（加上 1 个默认页面）
- **可自定义：** 截图、推广文本、App 预览、深层链接（iOS 18+）
- **关键词：** 每个关键词组合必须对单个 CPP 唯一
- **审核：** 独立于应用更新提交给 App Review
- **有机搜索：** CPP 自 2025 年 7 月起出现在有机搜索结果中
- **表现：** 平均转化率比默认页面高 2.5 个百分点

## 产品页面优化（A/B 测试）

- **变体：** 最多 3 个与原始版本对比
- **可测试：** App 图标、截图、App 预览视频
- **不可测试：** 标题、副标题、描述、关键词
- **并发测试：** 每个应用 1 个
- **最大持续时间：** 90 天
- **图标约束：** 所有图标变体必须在已发布的应用二进制文件中
- **置信度：** Apple 推荐 90% 阈值（贝叶斯方法）
- **启动后无法修改**测试

## 应用内活动

- **最大批准数量：** App Store Connect 中同时最多 15 个
- **最大发布数量：** App Store 上同时最多 10 个
- **最大持续时间：** 每个活动 31 天
- **活动前推广：** 开始前最多 14 天
- **徽章类型：** 挑战、竞赛、直播活动、重大更新、新赛季、首映、特别活动

**活动卡片图片：** 16:9，最小 1920x1080，最大 3840x2160
**活动详情图片：** 9:16，最小 1080x1920，最大 2160x3840

**不适合：** 重复的日常任务、没有新内容的价格促销、一般意识宣传活动。

## 评分与评论

- **SKStoreReviewController：** 每 365 天最多提示 3 次
- 系统控制显示频率（可能显示少于 3 次）
- 不要使用自定义按钮请求评论
- 开发者可以在 App Store Connect 中回复所有评论
- 摘要评分因地区而异

## 元数据拒绝触发器（App Review 指南）

| 指南 | 拒绝触发器                                                         |
| --------- | ------------------------------------------------------------------------- |
| 2.3.1     | 隐藏功能、误导性营销、虚假定价                      |
| 2.3.2     | 未在描述/截图中披露 IAP                            |
| 2.3.3     | 截图未显示应用使用中（仅闪屏/登录）                |
| 2.3.4     | 预览视频使用非应用内容                                      |
| 2.3.5     | 选择了错误的类别                                                   |
| 2.3.7     | 关键词堆砌：商标、竞争对手名称、定价、不相关术语 |
| 2.3.8     | 元数据不适合所有受众（必须评级为 4+）             |
| 2.3.10    | 元数据中出现其他平台名称/图像（Android 等）                  |
| 2.3.12    | 重大更改时使用通用的更新内容                                |
| 2.3.13    | 不准确的应用内活动元数据                                          |
