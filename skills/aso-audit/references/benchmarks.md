# ASO 基准与转化数据

来自 AppTweak、SplitMetrics、Sensor Tower 等机构的行业数据。更新于 2026 年 3 月。

## 按类别划分的转化率基准

**平均转化率（页面浏览到安装）：**

- iOS 整体：**25.0%**
- Google Play 整体：**27.3%**

| 类别          | iOS 转化率        | Google Play 转化率 |
| ----------------- | -------------- | --------------- |
| 导航        | 115%\*         | --              |
| 汽车与车辆   | --             | 70.5%           |
| 商务          | 66.7%          | --              |
| 音乐（游戏）     | --             | 45.0%           |
| 实用工具 | --             | 36.8%           |
| 购物          | --             | 27.7%           |
| 健康与健身  | --             | 23.2%           |
| 金融           | --             | 19.7%           |
| 餐饮      | --             | 13.1%           |
| 游戏（棋盘类）     | 1.2%           | 7.3%            |
| 游戏（整体）   | 3-5% 实际值 | --              |

\*超过 100% = 部分用户从搜索直接安装，未访问产品页面。

来源：AppTweak 2025 基准报告（2024 年上半年数据，美国市场）

## 评分对转化的影响

| 评分变化              | 转化影响                       |
| -------------------------- | --------------------------------------- |
| 3.0 到 4.0 星           | **+89%**                                |
| 4.0 到 4.5 星           | **+20-30%**                             |
| 4.3 到 4.6 星           | **+22-28%**（金融、健康）           |
| 与竞争对手相差 0.4 星 | **~25% 安装流失**（同一搜索） |
| 3 星对比 5 星应用       | **3 星应用转化减少 50%**    |

**关键阈值：**

- **4.0 星** = Apple 推荐、用户信任、转化可行性的最低要求
- **4.5+ 星** = 最佳区间。甜蜜点：4.1-4.9
- **5.0 星** 可能让用户觉得可疑
- **低于 3.5** = 两个商店的可见性急剧下降
- **79% 的用户**在下载前会查看评分
- **50% 的用户拒绝**低于 3 星的应用

来源：AppFollow、MobileAction、Sensor Tower、Troof.ai

## 预览视频的影响

**iOS：** +20-40% 转化提升（视频在产品页面自动播放）
**Google Play：** 提升有限（仅约 6% 的访客点击播放）

- iOS 11 引入自动播放后，转化**跃升 +47%**
- 观看视频的用户安装可能性**高出 2 倍**
- 平均观看时长：**4-6.5 秒**（前 5 秒至关重要）
- 超过 50% 的观众会看到结尾

**结论：** 视频在 iOS 上投资回报率高，在 Google Play 上投资回报率低。

来源：StoreMaven、SplitMetrics、Leanplum

## 截图的影响

- **90% 的用户**不会滚动到第 3 张截图之后
- 平均滚动率：仅 **17%**
- 用户在决定前花费 **6-10 秒**扫描
- **第一张截图决定一切**
- 精心设计的截图可提升转化 **20-35%**
- A/B 测试优胜者看到 **10-25% 的改进**
- **最佳数量：** 实用类应用 4-5 张，复杂应用 5-6 张
- 超过 6 张：收益递减，可能导致决策瘫痪
- 前 200 名应用每年更新截图 **2-4 次**
- 前 200 名 Google Play 游戏每年更新视觉素材**多达 8 次**
- **57% 的头部游戏**在 2024 年至少进行了 2 次截图 A/B 测试

来源：AppTweak、ASOMobile、Sensor Tower

## 自定义产品页面（Apple CPPs）

- 平均转化提升：**应用 +5.9%**，**游戏 +3.5%**
- 最佳案例：高达 **+8.6%**
- 有机引荐：**+2.5 个百分点**（相比 1.6% 基线提升 156%）
- Apple Ads CPP 转化率：**2024 年为 55.8%**（2023 年为 42.1%）
- **仅 31% 的应用**和 **26% 的游戏**使用 CPPs（采用率低 = 机会）
- 仅重新排序截图就在一个案例中带来 **+16.6% 的安装量**

来源：AppTweak、SplitMetrics、MobileAction

## 自定义商店列表（Google Play CSLs）

- 每个应用最多 **50 个自定义版本**
- 案例研究（Lockwood/Avakin Life）：2 个月内转化率 **+57%**
- 可针对不活跃/流失用户（28+ 天无活动）

来源：Phiture、MobileAction

## 应用内活动（Apple）

- **55% 的前 200 名应用**定期使用
- 编辑/浏览展示位带来 **+15-20% 更多展示次数**
- 一个案例：总展示次数**激增 +124%**
- 一个案例：**展示次数和首次下载均 +50%**
- 搜索转化率提升：**+10.3%**
- 重新下载增加：**+15.5%**
- **提升效果短暂** -- 活动结束后关键绩效指标回落到基线
- 最佳频率：**每月 2-4 个活跃活动**

来源：Phiture、AppTweak、Appalize

## 推广内容（Google Play）

- 获得推荐的应用看到**探索获取量翻倍**（Google 官方数据）
- 平均 28 日活跃用户 +2%，收入 +4%

来源：Google Play Console 文档

## A/B 测试影响阈值

| 改进幅度 | 分类                     |
| ----------- | ---------------------------------- |
| >10%        | 强力优胜 -- 立即应用 |
| 5-10%       | 显著优胜                  |
| 2-5%        | 边缘优胜                    |
| <2%         | 噪声 -- 不显著           |

来源：SplitMetrics、MobileAction
