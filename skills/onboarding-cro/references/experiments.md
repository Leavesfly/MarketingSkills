# 新用户引导实验创意

针对用户引导和激活的 A/B 测试与实验综合清单。

## 目录
- 流程简化实验（减少摩擦、步骤排序、进度与激励）
- 引导体验实验（产品导览、CTA 优化、UI 指引）
- 个性化实验（用户细分、动态内容）
- 快速获胜与参与度实验（价值实现时间、激励机制、支持与帮助）
- 邮件与多渠道实验（引导邮件、邮件内容、反馈循环）
- 重新激活实验（停滞用户挽回、回归体验）
- 技术与 UX 实验（性能、移动端引导、无障碍访问）
- 需要追踪的指标

## 流程简化实验

### 减少摩擦

| 测试 | 假设 |
|------|------------|
| 邮箱验证时机 | 引导过程中 vs. 引导完成后 |
| 空状态 vs. 模拟数据 | 预填充示例 |
| 预填充模板 | 使用模板加速设置 |
| OAuth 选项 | 更快的账户关联 |
| 必需步骤数量 | 减少必需步骤 |
| 可选与必填字段 | 最小化必填项 |
| 跳过选项 | 允许绕过非关键步骤 |

### 步骤排序

| 测试 | 假设 |
|------|------------|
| 步骤顺序 | 测试不同序列 |
| 价值优先排序 | 最高价值功能优先展示 |
| 摩擦点放置 | 将困难步骤后置 |
| 必填与可选平衡 | 必填步骤的比例 |
| 单一路径 vs. 分支路径 | 一条路径 vs. 个性化路径 |
| 快速开始 vs. 完整设置 | 通往价值的最小路径 |

### 进度与激励

| 测试 | 假设 |
|------|------------|
| 进度条 | 显示完成百分比 |
| 清单长度 | 3-5 项 vs. 5-7 项 |
| 游戏化元素 | 徽章、奖励、成就 |
| 完成度提示 | "X% 已完成"可见性 |
| 起始点设置 | 从 20% 开始 vs. 从 0% 开始 |
| 庆祝时刻 | 确认完成节点 |

## 需要追踪的指标

对于所有实验，测量：

| 指标 | 描述 |
|--------|-------------|
| 激活率 | 达到激活事件的百分比 |
| 激活时间 | 首次获得价值所需的小时数/天数 |
| 步骤完成率 | 完成每个步骤的百分比 |
| 流失点 | 用户放弃的位置 |
| 回访率 | 回来的用户 |
| 第 1/7/30 天留存率 | 随时间的参与度 |
| 功能采用率 | 哪些功能被使用 |
| 支持请求量 | 引导期间的数量 |
