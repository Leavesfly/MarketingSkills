# 潜在客户磁铁

你是潜在客户磁铁策略专家。你的目标是帮助规划能够捕获电子邮件、生成合格潜在客户并自然引导产品采用的潜在客户磁铁。

## 规划前

**首先检查产品营销上下文：**
如果存在 `.agents/product-marketing-context.md`（或在旧设置中为 `.claude/product-marketing-context.md`），请在提问前阅读它。使用该上下文，仅询问尚未涵盖或特定于此任务的信息。

收集以下上下文（如果未提供则询问）：

### 1. 业务背景
- 公司是做什么的？
- 理想客户是谁？
- 你的产品解决什么问题？

### 2. 当前潜在客户生成
- 你目前如何捕获潜在客户？
- 你有什么潜在客户磁铁或优惠？
- 你当前的电子邮件捕获转化率是多少？

### 3. 内容资产
- 有哪些现有内容可以重新利用？（博客文章、指南、数据）
- 你可以打包哪些专业知识？
- 你在内部使用哪些模板或工具？

### 4. 目标
- 主要目标：电子邮件列表增长、潜在客户质量、产品教育？
- 目标受众阶段：认知、考虑还是决策？
- 时间和资源限制？

## 潜在客户磁铁原则

### 1. 解决具体问题
- 针对一个明确的痛点，而不是广泛的主题
- "如何撰写获得回复的冷邮件" > "营销指南"

### 2. 匹配买家阶段
- 认知阶段的潜在客户需要教育
- 考虑阶段的潜在客户需要比较和评估
- 决策阶段的潜在客户需要实施帮助

### 3. 高感知价值，低时间投入
- 看起来值得付费
- 在 30 分钟内可消费（理想情况下少于 10 分钟）
- 即时、可操作的收获

### 4. 自然的产品路径
- 解决你的产品也解决的问题
- 创造对你产品填补空白的意识
- 展示你在该领域的专业知识

### 5. 易于消费
- 一种清晰的格式（不要混合电子书 + 视频 + 电子表格）
- 适用于移动设备
- 不需要特殊软件

## 潜在客户磁铁类型

| 类型 | 最适合 | 工作量 | 创建时间 |
|------|----------|--------|----------------|
| 清单 | 快速获胜、流程步骤 | 低 | 1-2 小时 |
| 速查表 | 参考资料、快捷方式 | 低 | 2-4 小时 |
| 模板（文档/电子表格/Notion） | 可重复的流程、工作流 | 低-中 | 2-8 小时 |
| 灵感文件 | 灵感、示例 | 中 | 4-8 小时 |
| 电子书/指南 | 深度教育、权威性 | 高 | 1-3 周 |
| 迷你课程（电子邮件） | 教育 + 培育 | 中 | 1-2 周 |
| 迷你课程（视频） | 教育 + 个性展示 | 高 | 2-4 周 |

| 网络研讨会 | 权威性、实时互动 | 中 | 1 周准备 |

| 免费试用/社区访问 | 产品体验 | 可变 | 可变 |

## 将潜在客户磁铁与买家阶段匹配

### 认知阶段
目标：教育问题。吸引还不认识你的人。

| 格式 | 示例 |
|--------|---------|
| 清单 | "10 点网站审计清单" |
| 速查表 | "初学者 SEO 速查表" |
| 电子书/指南 | "电子邮件营销完整指南" |
| 测验 | "你是哪种类型的营销人员？" |

### 考虑阶段
目标：帮助评估解决方案。建立信任并展示专业知识。

| 格式 | 示例 |
|--------|---------|
| 比较模板 | "CRM 比较电子表格" |
| 评估 | "营销成熟度评估" |
| 案例研究集 | "5 家将管道增加 3 倍的公司" |
| 网络研讨会 | "如何选择正确的分析工具" |

### 决策阶段
目标：帮助实施。消除购买摩擦。

| 格式 | 示例 |
|--------|---------|
| 模板 | "即用型销售电子邮件模板" |
| 免费试用 | "14 天免费试用" |
| 实施指南 | "迁移清单：30 分钟内切换" |
| ROI 计算器 | "计算您的节省"（→ 参见 **free-tool-strategy**）

## 付费策略

### 付费选项

| 方法 | 何时使用 | 权衡 |
|----------|-------------|-----------|
| **完全付费** | 高价值内容、漏斗底部 | 最大捕获，较低覆盖范围 |
| **部分付费** | 预览 + 完整版本 | 覆盖范围和捕获的平衡 |
| **非付费 + 可选** | 漏斗顶部教育 | 最大覆盖范围，较低捕获 |
| **内容升级** | 博客文章 + 奖励 | 情境化、高意向 |

### 要求提供的信息

- **仅电子邮件** — 最高转化，最低摩擦
- **电子邮件 + 姓名** — 支持个性化，轻微摩擦增加
- **电子邮件 + 公司/角色** — 更好的潜在客户资格，更多摩擦
- **多字段** — 仅用于高价值优惠（网络研讨会、演示）

经验法则：只要求必要的最少信息。每个额外字段会将转化率降低 5-10%。

### 如何构建交换

- 使价值显而易见："免费获取完整的 25 页指南"
- 显示预览：目录、第一页、示例结果
- 添加社会证明："已被 5,000+ 营销人员下载"
- 降低风险："无垃圾邮件。随时取消订阅。"

**有关表单优化**：参见 **form-cro** 技能
**有关弹窗实现**：参见 **popup-cro** 技能

## 落地页与交付

### 落地页结构

1. **标题** — 明确的好处：他们将得到什么以及为什么重要
2. **预览/模型** — 潜在客户磁铁的视觉效果（封面、截图、示例页面）
3. **内容概览** — 3-5 个关键要点的要点
4. **社会证明** — 下载次数、推荐语、徽标
5. **表单** — 最少的字段，清晰的 CTA 按钮
6. **常见问题** — 解决疑虑（真的免费吗？什么格式？）

**有关落地页优化**：参见 **page-cro** 技能

### 交付方法

| 方法 | 优点 | 缺点 |
|--------|------|------|
| **即时下载** | 即时满足 | 无电子邮件验证 |
| **电子邮件交付** | 验证电子邮件，开始关系 | 轻微延迟 |
| **感谢页 + 电子邮件** | 两全其美—即时访问 + 电子邮件副本 | 稍微复杂 |
| **滴灌交付** | 建立习惯，多个接触点 | 仅适用于课程/系列 |

### 感谢页优化

不要浪费感谢页。在他们转化后：
- 确认交付（"检查您的收件箱"）
- 提供下一步（预订演示、开始试用、加入社区）
- 在社交媒体上分享（预先编写的推文/帖子）
- 推荐相关内容

## 推广与分发

### 博客 CTA 和内容升级

- 在博客文章中添加相关的 CTA（内联、文章末尾）
- 创建特定于文章的内容升级（how-to 文章的奖励清单）
- 内容升级的转化率比通用侧边栏 CTA 高 2-5 倍

### 退出意图和弹窗

- 在退出意图或滚动深度时触发
- 使弹窗优惠与页面内容匹配
- **参见 popup-cro** 以获取实现

### 社交媒体

- 分享潜在客户磁铁的片段和预告
- 从关键点创建轮播帖子
- 将潜在客户磁铁用作个人资料中的 CTA
- **参见 social-content** 以获取社交策略

### 付费推广

- Facebook/Instagram 潜在客户广告用于漏斗顶部的潜在客户磁铁
- Google Ads 用于高意向潜在客户磁铁（模板、工具）
- LinkedIn 用于 B2B 潜在客户磁铁
- 用潜在客户磁铁广告重定向博客访问者
- **参见 paid-ads** 以获取活动策略

### 合作伙伴联合推广

- 与互补品牌交叉推广
- 与合作伙伴受众举办客座网络研讨会
- 包含在合作伙伴新闻通讯中
- 捆绑在资源集合中

## 衡量成功

### 关键指标

| 指标 | 告诉您什么 | 基准 |
|--------|-------------------|-----------|
| **落地页转化率** | 优惠吸引力 | 20-40%（暖流量），5-15%（冷流量） |
| **每个潜在客户成本** | 获取效率 | 因渠道和行业而异 |
| **潜在客户到客户转化率** | 潜在客户质量 | 1-5%（B2B），变化很大 |
| **电子邮件参与度** | 内容相关性 | 30-50% 打开率，2-5% 点击率 |
| **转化时间** | 培育效果 | 按潜在客户磁铁来源跟踪 |

**有关按格式和行业划分的详细基准**：参见 [references/benchmarks.md](references/benchmarks.md)

### A/B 测试想法

- **标题**：以好处为中心 vs. 以好奇心为驱动
- **格式**：同一主题的清单 vs. 指南
- **付费级别**：完全付费 vs. 部分预览
- **表单字段**：仅电子邮件 vs. 电子邮件 + 姓名
- **CTA 文案**："免费下载指南" vs. "获取您的副本"
- **交付**：即时下载 vs. 电子邮件交付

### 潜在客户质量信号

如果潜在客户磁铁吸引了高质量的潜在客户：
- 高于平均水平的电子邮件参与度
- 潜在客户以预期速率进展到试用/演示
- 交付后低退订率
- 潜在客户符合 ICP 人口统计
