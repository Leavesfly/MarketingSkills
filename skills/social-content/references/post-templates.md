# 帖子格式模板

适用于不同平台和内容类型的即用型模板。

## 目录
- LinkedIn 帖子模板（故事帖、反传统观点帖、列表帖、操作指南帖）
- Twitter/X 线程模板（教程线程、故事线程、拆解线程）
- Instagram 模板（轮播图钩子、Reel 脚本）
- 钩子公式（好奇心钩子、故事钩子、价值钩子、反传统钩子、社会证明钩子）

## LinkedIn 帖子模板

### 故事帖
```
[钩子：意想不到的结果或教训]

[场景设置：何时/何地发生]

[你面临的挑战]

[你尝试的方法 / 发生的事情]

[转折点]

[结果]

[给读者的教训]

[引发互动的问题]
```

### 反传统观点帖
```
[大胆陈述的不受欢迎观点]

原因如下：

[原因 1]
[原因 2]
[原因 3]

[你推荐的做法]

[邀请讨论："我错了吗？"]
```

### 列表帖
```
在[可信度建立者]之后，我关于[主题]学到的 X 件事：

1. [要点] — [简要说明]

2. [要点] — [简要说明]

3. [要点] — [简要说明]

[总结性洞察]

哪一点与你最有共鸣？
```

### 操作指南帖
```
如何在[时间框架]内[达成结果]：

步骤 1：[行动]
↳ [为什么这很重要]

步骤 2：[行动]
↳ [关键细节]

步骤 3：[行动]
↳ [要避免的常见错误]

[你可以预期的结果]

[CTA 或问题]
```

## Twitter/X 线程模板

### 教程线程
```
推文 1：[钩子 + 价值承诺]

"以下是如何[结果]的确切方法（分步）："

推文 2-7：[每条推文一个步骤，附带详情]

最后一条推文：[总结 + CTA]

"如果这对你有帮助，请关注我以获取更多关于[主题]的内容"
```

### 故事线程
```
推文 1：[引人入胜的钩子]

"[时间]前，[意想不到的事情发生了]。以下是完整故事："

推文 2-6：[故事节拍，构建张力]

推文 7：[结局和教训]

最后一条推文：[收获 + 互动请求]
```

### 拆解线程
```
推文 1：[公司/人物]刚刚[做了某事]。

以下是为什么这很聪明（以及你能学到什么）：

推文 2-6：[分析要点]

推文 7：[你的关键收获]

"[相关洞察 + 关注 CTA]"
```

## Instagram 模板

### 轮播图钩子
```
[幻灯片 1：大胆的陈述或问题]
[幻灯片 2-9：每张幻灯片一个要点，视觉 + 文字]
[幻灯片 10：总结 + CTA]

标题：[展开主题，添加上下文，包含 CTA]
```

### Reel 脚本
```
钩子（0-2 秒）：[模式中断或大胆声明]
铺垫（2-5 秒）：[技巧的背景]
价值（5-25 秒）：[实际建议/内容]
CTA（25-30 秒）：[关注、评论、分享、链接]
```

## 钩子公式

第一行决定了是否有人会阅读其余内容。

### 好奇心钩子
- "我对[常见信念]的看法是错误的。"
- "[结果]发生的真正原因并非你所想。"
- "[令人印象深刻的结果]——而且只用了[短得惊人的时间]。"
- "没人谈论[内部知识]。"

### 故事钩子
- "上周，[意想不到的事情]发生了。"
- "我差点[重大错误/失败]。"
- "3 年前，我[过去状态]。如今，[当前状态]。"
- "[人物]告诉了我一些我永远不会忘记的事情。"

### 价值钩子
- "如何[理想结果]（无需[常见痛点]）："
- "[数量]个[事物]可[结果]："
- "[结果]的最简单方法："
- "停止[常见错误]。改为这样做："

### 反传统钩子
- "不受欢迎的观点：[大胆的陈述]"
- "[常见建议]是错误的。原因如下："
- "我停止了[常见做法]，然后[积极结果]。"
- "每个人都说[X]。真相是[Y]。"

### 社会证明钩子
- "我们在[时间框架]内[达成结果]。以下是完整故事："
- "[数量]个人问我关于[主题]的问题。以下是我的回答："
- "[权威人物]教会了我[教训]。"
