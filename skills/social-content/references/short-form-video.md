# 短视频：钩子、脚本与策略

在 TikTok、Instagram Reels 和 YouTube Shorts 上创建短视频内容的详细参考。

---

## 视频钩子库

### 好奇心钩子（最适合互动）

**"秘密"公式：**
- "没人谈论的[结果]秘诀"
- "我发现了[产品/平台]中改变一切的隐藏功能"
- "我不敢相信这真的有效..."

**意外发现：**
- "我尝试了[事物]30天，结果完全出乎意料"
- "这彻底改变了我对[主题]的看法"
- "没人谈论这个，但是..."

**提问式：**
- "为什么没人谈论这个？"
- "我是唯一一个不知道这个的人吗？"
- "[常见事物]不起作用的原因是..."

### 价值钩子（最适合收藏）

**承诺式：**
- "如何在[特定时间框架]内[达成结果]"
- "[数量]个将[受益]的[事物]"
- "60秒内了解关于[主题]的一切"

**技巧/捷径：**
- "真正有效的[结果]技巧"
- "[形容词]的[结果]方式"
- "如果你在[问题]上挣扎，看这个"

**警告式：**
- "停止做[常见做法]——原因如下"
- "[数量]个正在扼杀你[结果]的错误"
- "为什么[你认为好的事物]实际上在伤害你"

### 故事钩子（最适合观看时长）

**转变式：**
- "3个月前，我[糟糕状态]。今天，我[良好状态]。"
- "这是我如何从[之前]到[之后]的过程"
- "我曾经认为[旧信念]。然后[事件]改变了一切。"

**失败式：**
- "我在[主题]上犯了一个巨大的错误"
- "这就是我为什么停止[常见做法]"
- "因为这个错误，我失去了[重要的东西]"

**旅程式：**
- "所以刚刚发生了这件事..."
- "这改变了我的一切"
- "让我告诉你那次我[有趣的情况]的故事"

### 争议性钩子（最适合评论）

- "不受欢迎的观点：[大胆的陈述]"
- "[常见建议]实际上是错误的"
- "我会因此被骂，但是..."
- "大多数[受众]完全搞错了这一点"

## 脚本模板

```markdown
## 视频：[工作标题]

**平台：** TikTok / Reels / Shorts
**长度：** XX 秒
**格式：** [面对镜头 / 幻灯片 / 演示 / 屏幕录制]

### 钩子（0-3秒）
- 视觉：[观众看到的]
- 音频：[他们听到的]
- 文字叠加：[屏幕上的文字]

### 主体（3-X秒）
- [时间戳] - [发生什么/你说什么]
- [时间戳] - [下一个节拍]
- [继续...]

### CTA（最后3-5秒）
- 口头：[你说的]
- 文字：[屏幕上的文字]
- 行动：[关注、评论、链接在简介等]

### 制作说明
- 音乐/音效：[热门音效或音乐选择]
- 需要的 B-roll：[列出需要的任何片段]
- 图形：[任何文字动画或叠加]
```

## 其他视频结构

### 故事弧线（45-60秒）

```
[0-3秒]  钩子：预告结果
[3-15秒] 铺垫：背景和利害关系
[15-45秒] 旅程：发生了什么
[45-55秒] 结局：结果
[55-60秒] 教训/CTA
```

最适合：个人故事、案例研究、推荐语

### POV/短剧（15-30秒）

```
[0-3秒]  设置：文字叠加设定场景
[3-25秒] 表演：演绎相关场景
[25-30秒] 笑点或转折
```

最适合：相关内容、幽默、小众社区

## 视觉模式

### 面对镜头
- 良好的照明（环形灯或窗户光）
- 与相机眼神接触
- 用手势强调
- 有趣的背景（书架、植物、工作室设置）

### 幻灯片/轮播视频
- 每张幻灯片都有强烈的视觉效果（每张2-4秒）
- 带有要点的文字叠加
- 一致的风格/品牌
- 配音或热门音效

### 屏幕录制
- 放大重要区域
- 添加光标高亮或点击动画
- 保持移动平滑且有意图
- 在角落叠加你的脸（可选但提升互动）

### 大量 B-roll
- 展示而非讲述
- 快速剪辑（每个镜头1-3秒）
- 将剪辑匹配到配音节拍
- 混合广角、中景和特写镜头

## 音频策略

### 何时使用热门音效
- 娱乐/生活方式内容，其中音效适合你的信息
- 当趋势仍在上升时（检查平台热门页面）
- 当它分散对你的信息的注意力或已经 declining 时不要使用

### 何时使用原创音频
- 你在说话的教育内容
- 故事时间和个人叙述
- 产品演示和教程
- 建立可识别的品牌声音

### 配音技巧
- 说话速度略快于正常对话
- 变化你的语调——避免单调 delivery
- 在关键点上暂停以强调
- 在安静的空间录制，使用降噪
- AI 声音适用于无脸内容（ElevenLabs 等）

### 音乐选择
- 将能量与内容匹配（提示用 upbeat，故事用 emotional）
- 在 Reels/Shorts 上避免版权音乐
- 使用平台音乐库以确保安全
- 在配音下降低音乐音量（ducking）

## 发布策略

### 最佳发布时间（测试你的受众）

| 平台 | 最佳时间（本地） |
|----------|-------------------|
| TikTok | 7-9 AM, 12-3 PM, 7-11 PM |
| Reels | 9 AM, 12 PM, 7-9 PM |
| Shorts | 12-3 PM, 7-9 PM |

### 频率建议

| 目标 | 最低 | 最佳 |
|------|---------|---------|
| 增长 | 1/天 | 2-4/天 |
| 维持 | 3/周 | 1/天 |
| 测试 | 2/周 | 5/周 |

### 批量创建工作流

1. **构思**（30分钟）：生成10-20个概念
2. **脚本**（1小时）：为5-10个视频编写脚本
3. **批量拍摄**（2小时）：录制所有面对镜头内容
4. **编辑**（2-3小时）：编辑并添加字幕
5. **排期**（30分钟）：排队到最佳时间

## 分析与迭代

### 重要指标

| 指标 | 它告诉你什么 |
|--------|-------------------|
| 观看时长% | 内容在整个过程中是否吸引人？ |
| 完成率 | 钩子+内容是否交付？ |
| 收藏 | 内容是否有价值到值得 revisit？ |
| 分享 | 内容是否值得传播？ |
| 评论 | 内容是否引发了对话？ |
| 关注 | 观众是否想要更多你的内容？ |

### 测试内容

1. **钩子**：相同内容，不同开头
2. **长度**：15秒 vs 30秒 vs 60秒
3. **格式**：面对镜头 vs 幻灯片 vs 演示
4. **时间**：早上 vs 下午 vs 晚上
5. **CTA**：不同的行动号召

### 何时调整方向

- 5+个视频完成率 <1% → 改变钩子
- 高观看量但低关注 → 检查 CTA 和内容-受众匹配
- 高收藏但低分享 → 内容有价值但不社交
- 很多评论但负面 → 利用争议或调整语气
