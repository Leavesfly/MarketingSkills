# 主题行优化

主题行决定邮件是否被阅读。数据反直觉：**简短、无聊、看起来像内部邮件的主题行 decisively 胜出。**

## 长度：2–4 个词

- 2 个词的主题行比 5 个词的获得 **60% 更多的打开**（Lavender）。
- 从 2 个词增加到 4 个词会使回复减少 **17.5%**。
- 2–4 个词产生 **46% 的打开率**，而 10 个词为 34%（Belkins，550 万封邮件）。
- 移动设备在 30–35 个字符处截断——简洁是实际必要。

## 内部伪装原则

看起来像来自同事而非供应商的主题行使打开率翻倍（Gong）。买家在打开前会进行心理分类——如果看起来像销售，就会被过滤。

**高绩效示例：** "reply rates" · "trial delays" · "hiring ops" · "employee turnover" · "Q2 forecast" · "new patients" · "personalization issue" · "second page"

## 大写：小写胜出

全小写具有最高的打开率（Gong，8500 万+ 邮件）。小写看起来更个人化/内部化。对于冷外联，小写优于标题大小写。

## 个性化：上下文胜过名字

个性化主题行将打开率提高 **26–50%**，但类型很重要：

- **主题行中的名字 → 回复减少 12%。** 表明是自动化。
- **上下文个性化有效：** 痛点、竞争对手、触发事件、行业挑战。
- 使用 {{painPoint}}、{{competitor}}、{{commonGround}} —— 而不是 {{firstName}}。

## 问句：仅在高度具体时使用

数据冲突：Belkins 表示问句表现良好（46% 打开率）。Lavender 表示问句使打开率降低 **56%**。解决方案：**具体的痛点问句有效**（"Need help with {{challenge}}?"），**通用问句失败**（"Quick question?" / "Have 15 minutes?"）。默认使用陈述句。

## 避免什么

| 反面模式                                   | 影响                      |
| ---------------------------------------------- | --------------------------- |
| 销售语言（"increase"、"boost"、"ROI"）   | -17.9% 打开                |
| 紧急词汇（"ASAP"、"urgent"）               | 低于 36% 打开             |
| 过度标点（"!!!" 或 "??"）          | -36% 打开                  |
| 数字和百分比                        | -46% 打开                  |
| 表情符号                                         | 损害 B2B 专业性    |
| 在主题中推销产品                    | -57% 回复                |
| 空/无主题行                          | +30% 打开但 -12% 回复 |
| 垃圾邮件触发词（"free"、"guarantee"、"act now"） | 可送达性风险         |

## C-Suite 主题行

高管每天收到 300–400 封邮件，在几秒钟内做出决定。当邮件通过他们的过滤时，他们的回复频率比非 C-Suite 高 **23%**（6.4% 回复率）。

有效的方法：超简洁、人性化、低调。"{{companyInitiative}}" · "thank you" · "an update" · "a question" · 引用特定项目或触发事件。

任何"销售味"的东西都会被立即拒绝。
