# 页面 CRO 实验创意

按页面类型组织的 A/B 测试和实验综合清单。

## 目录
- 首页实验（Hero 区域、信任与社会证明、功能与内容、导航与 UX）
- 定价页实验（价格展示、定价 UX、异议处理、信任信号）
- 演示请求页实验（表单优化、页面内容、CTA 与路由）
- 资源/博客页面实验（内容 CTA、资源部分）
- 落地页实验（信息匹配、转化聚焦、页面长度）
- 功能页实验（功能展示、转化路径）
- 跨页面实验（全站测试、导航测试）

## 首页实验

### Hero 区域

| 测试 | 假设 |
|------|------------|
| 标题变体 | 具体 vs. 抽象信息 |
| 副标题清晰度 | 添加/优化以支持标题 |
| 首屏 CTA | 包含或排除醒目的 CTA |
| Hero 视觉格式 | 截图 vs. GIF vs. 插图 vs. 视频 |
| CTA 按钮颜色 | 测试对比度和可见性 |
| CTA 按钮文案 | "开始免费试用" vs. "立即开始" vs. "查看演示" |
| 交互式演示 | 用产品立即吸引访客 |

### 信任与社会证明

| 测试 | 假设 |
|------|------------|
| 徽标放置 | Hero 区域 vs. 首屏以下 |
| Hero 中的案例研究 | 立即展示结果 |
| 信任徽章 | 添加安全、合规、奖项 |
| 标题中的社会证明 | "加入 10,000+ 团队"信息 |
| 推荐语位置 | 首屏以上 vs. 专用部分 |
| 视频推荐语 | 比文字引用更具吸引力 |

### 功能与内容

| 测试 | 假设 |
|------|------------|
| 功能展示 | 图标 + 描述 vs. 详细部分 |
| 部分顺序 | 将高价值功能上移 |
| 次要 CTA | 在整个页面中添加/移除 |
| 好处 vs. 功能焦点 | 以结果为导向 |
| 比较部分 | 与竞争对手或现状对比 |

### 导航与 UX

| 测试 | 假设 |
|------|------------|
| 固定导航 | 带有 CTA 的持久导航 |
| 导航菜单顺序 | 高优先级项目放在边缘 |
| 导航 CTA 按钮 | 在导航中添加醒目按钮 |
| 支持小部件 | 实时聊天 vs. AI 聊天机器人 |
| 页脚优化 | 更清晰的次要转化 |
| 退出意图弹窗 | 捕获放弃访问的访客 |

## 定价页实验

### 价格展示

| 测试 | 假设 |
|------|------------|
| 年度 vs. 月度显示 | 突出节省或简化 |
| 价格点 | $99 vs. $100 vs. $97 心理学 |
| "最受欢迎"徽章 | 突出目标计划 |
| 层级数量 | 3 vs. 4 vs. 2 个可见选项 |
| 价格锚定 | 排序计划以锚定期望 |
| 自定义企业层级 | 显示 vs. "联系销售" |

### 定价 UX

| 测试 | 假设 |
|------|------------|
| 定价计算器 | 用于基于用量的定价清晰度 |
| 引导式定价流程 | 多步骤向导 vs. 比较表 |
| 功能比较格式 | 表格 vs. 可展开部分 |
| 月/年切换 | 突出显示节省 |
| 计划推荐测验 | 帮助访客选择 |
| 结账流程长度 | 选择计划后所需的步骤 |

### 异议处理

| 测试 | 假设 |
|------|------------|
| FAQ 部分 | 解决定价异议 |
| ROI 计算器 | 展示价值 vs. 成本 |
| 退款保证 | 醒目放置 |
| 每人细分 | 团队计划的清晰度 |
| 功能包含清晰度 | 每个层级包含什么 |
| 竞争对手比较 | 并排价值比较 |

### 信任信号

| 测试 | 假设 |
|------|------------|
| 价值推荐语 | 专门关于 ROI 的引用 |
| 客户徽标 | 靠近定价部分 |
| 评分 | G2/Capterra 评级 |
| 案例研究摘要 | 具体的定价/价值结果 |

## 演示请求页实验

### 表单优化

| 测试 | 假设 |
|------|------------|
| 字段数量 | 更少字段，更高完成率 |
| 多步骤 vs. 单步 | 进度条鼓励 |
| 表单位置 | 首屏以上 vs. 内容之后 |
| 电话字段 | 包含 vs. 排除 |
| 字段丰富化 | 隐藏可以自动填充的字段 |
| 表单标签 | 字段内 vs. 上方 |

### 页面内容

| 测试 | 假设 |
|------|------------|
| 表单上方的好处 | 在询问之前强化价值 |
| 演示预览 | 展示演示体验的视频/GIF |
| "你将学到什么" | 清晰设定期望 |
| 表单附近的推荐语 | 减少决策点的摩擦 |
| 表单下方的 FAQ | 解决常见异议 |
| 视频 vs. 文字 | 解释价值的格式 |

### CTA 与路由

| 测试 | 假设 |
|------|------------|
| CTA 文案 | "预订你的演示" vs. "安排 15 分钟通话" |
| 按需选项 | 即时演示与实时选项并列 |
| 个性化信息 | 基于访客数据/来源 |
| 导航移除 | 减少页面干扰 |
| 日历集成 | 内联预订 vs. 外部链接 |
| 资格路由 | 某些自助服务，其他销售跟进 |

## 落地页实验

### 信息匹配

| 测试 | 假设 |
|------|------------|
| 标题匹配 | 完全匹配广告文案 |
| 视觉匹配 | 匹配广告创意 |
| 优惠对齐 | 与广告承诺的相同优惠 |
| 受众特定页面 | 每个细分市场的不同页面 |

### 转化聚焦

| 测试 | 假设 |
|------|------------|
| 导航移除 | 单一焦点页面 |
| CTA 重复 | 整个页面中的多个 CTA |
| 表单 vs. 按钮 | 直接捕获 vs. 点击通过 |
| 紧迫感/稀缺性 | 如果真实，测试信息 |
| 社会证明密度 | 数量和放置 |
| 视频包含 | 用视频解释优惠 |

### 页面长度

| 测试 | 假设 |
|------|------------|
| 短 vs. 长 | 快速转化 vs. 完整论证 |
| 仅首屏 | 所需滚动最少 |
| 部分顺序 | 最重要的内容优先 |
| 页脚移除 | 消除导航 |
