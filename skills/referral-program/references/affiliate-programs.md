## 佣金结构

**销售百分比：**
- 标准：首次销售或第一年的 10-30%
- 适用于：电子商务、具有明确定价的 SaaS
- 示例："赚取你推荐的每笔销售的 25%"

**每次操作的固定费用：**
- 标准：根据价值 $5-500
- 适用于：潜在客户生成、试用、免费增值
- 示例："每个合格的演示 $50"

**经常性佣金：**
- 标准：经常性收入的 10-25%
- 适用于：订阅产品
- 示例："12 个月内订阅费的 20%"

**分层佣金：**
- 适用于：激励高绩效者
- 示例："1-10 次销售 20%，11-25 次 25%，26 次以上 30%"

点击后多长时间内联盟成员获得归因？

| 持续时间 | 使用场景 |
|----------|----------|
| 24 小时 | 高体量、低考虑度购买 |
| 7-14 天 | 标准电子商务 |
| 30 天 | 标准 SaaS/B2B |
| 60-90 天 | 长销售周期、企业级 |
| 终身 | 高级联盟关系 |

## 联盟成员招募

### 在哪里寻找联盟成员：
- 创建内容的现有客户
- 行业博主和评论者
- 你所在领域的 YouTuber
- 新闻通讯作者
- 互补工具公司
- 顾问和代理机构

### 外联模板：
```
主题：合作机会 — [Your Product]

你好 [Name]，

我一直在关注你关于 [topic] 的内容——特别是 [specific piece]——我认为可能存在很好的合作契合点。

[Your Product] 帮助 [audience] [achieve outcome]，我认为你的受众会觉得它很有价值。

我们为合作伙伴提供 [commission structure]，以及 [additional benefits: early access, co-marketing, etc.]。

你愿意了解更多吗？

[Your name]
```

## 联盟成员赋能

为联盟成员提供：
- [ ] 唯一跟踪链接/代码
- [ ] 产品概述和关键优势
- [ ] 目标受众描述
- [ ] 与竞争对手的比较
- [ ] 创意素材（徽标、横幅、图片）
- [ ] 示例文案和要点
- [ ] 案例研究和推荐语
- [ ] 演示访问或免费账户
- [ ] FAQ 和异议处理
- [ ] 付款条款和时间表

## 工具与平台

### 推荐计划工具

**功能齐全的平台：**
- ReferralCandy — 专注于电子商务
- Ambassador — 企业级推荐计划
- Friendbuy — 电子商务和订阅

**联盟网络：**
- ShareASale — 大型商户网络

- Tapfiliate — 简单的 SaaS 联盟跟踪

## 欺诈预防

### 常见推荐欺诈
- 自我推荐（创建虚假账户）
- 推荐环（群体互相推荐）
- 优惠券网站发布推荐码
- 虚假电子邮件地址
- VPN/设备欺骗

### 预防措施

**技术层面：**
- 需要电子邮件验证
- 设备指纹识别
- IP 地址监控
- 延迟奖励支付（激活后）
- 最低活动阈值

**政策层面：**
- 清晰的服务条款
- 每个时期的最大推荐数
- 退款/拒付的奖励追回
- 对可疑模式进行人工审查

**结构层面：**
- 要求被推荐用户采取有意义的行动
- 限制终身奖励上限
- 以产品积分形式支付奖励（对欺诈者吸引力较小）
