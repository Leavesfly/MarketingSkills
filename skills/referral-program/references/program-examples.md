## Dropbox（经典案例）

**计划：** 赠送 500MB 存储空间，获得 500MB 存储空间

**成功原因：**
- 奖励与产品价值直接相关
- 低摩擦（只需一个电子邮件地址）
- 双方平等受益
- 通过进度跟踪实现游戏化

## Uber/Lyft

**计划：** 赠送 $10 乘车积分，当他们乘车时获得 $10

**成功原因：**
- 即时、清晰的价值
- 双边激励
- 易于分享（代码/链接）
- 在自然时刻触发

---

## Morning Brew

**计划：** 针对订阅者推荐的分层奖励
- 3 次推荐：新闻通讯贴纸
- 5 次推荐：T 恤
- 10 次推荐：马克杯
- 25 次推荐：连帽衫

**成功原因：**
- 游戏化驱动持续参与
- 实物奖励可分享（带来更多推荐）
- 相对于订阅者价值成本低
- 建立状态/身份认同

---

## Notion

**计划：** 每次推荐 $10 积分（教育）

**成功原因：**
- 针对高分享受众（学生）
- 产品在团队中自然传播
- 积分让用户保持参与

## 激励类型比较

| 类型 | 优点 | 缺点 | 最适合 |
|------|------|------|----------|
| 现金/积分 | 普遍有价值 | 感觉交易化 | 市场平台、金融科技 |
| 产品积分 | 推动使用 | 仅在他们使用时才有价值 | SaaS、订阅 |
| 免费月份 | 清晰的价值 | 可能吸引免费获得者 | 订阅产品 |
| 功能解锁 | 对你来说成本低 | 仅适用于 gated 功能 | 免费增值产品 |
| 周边商品/礼品 | 令人难忘、可分享 | 物流复杂性 | 以品牌为中心的公司 |
| 慈善捐赠 | 感觉良好 | 个人动机较低 | 使命驱动型品牌 |

## 激励规模框架

**计算你的最大激励：**
```
Max Referral Reward = (Customer LTV × Gross Margin) - Target CAC
```

**示例：**
- LTV：$1,200
- 毛利率：70%
- 目标 CAC：$200
- 最大奖励：($1,200 × 0.70) - $200 = $640

**典型推荐奖励：**
- B2C：$10-50 或首次购买的 10-25%
- B2B SaaS：$50-500 或 1-3 个月免费
- 企业级：更高，通常定制

## 病毒系数与指标

### 关键指标

**病毒系数（K-factor）：**
```
K = 邀请数 × 转化率

K > 1 = 病毒式增长（每个用户带来超过 1 个新用户）
K < 1 = 放大增长（推荐补充其他获客渠道）
```

**示例：**
- 平均客户发送 3 次邀请
- 15% 的邀请转化
- K = 3 × 0.15 = 0.45

**推荐率：**
```
Referral Rate = (推荐的客户) / (总客户)
```

基准：
- 良好：10-25% 的客户推荐
- 优秀：25-50%
- 卓越：50%+

**每个推荐人的推荐数：**

基准：
- 平均：每个推荐人 1-2 次推荐
- 良好：2-5
- 卓越：5+

### 计算推荐计划 ROI

```
Referral Program ROI = (来自被推荐客户的收入 - 计划成本) / 计划成本

计划成本 = 支付的奖励 + 工具成本 + 管理时间
```

**单独跟踪：**
- 每个被推荐客户的成本（通过推荐的 CAC）
- 被推荐客户的 LTV（通常高于平均水平）
- 推荐奖励的回收期
