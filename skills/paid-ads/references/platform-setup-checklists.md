# 平台设置清单

主要广告平台的完整设置清单。

## 目录
- Google Ads 设置（账户基础、转化跟踪、Analytics 集成、受众设置、广告系列准备、广告扩展、品牌保护）
- Meta Ads 设置（Business Manager 基础、像素和跟踪、域名和聚合事件、受众设置、商品目录、创意素材、合规性）
- LinkedIn Ads 设置（Campaign Manager 基础、Insight Tag 和跟踪、受众设置、Lead Gen Forms、Document Ads、创意素材、预算考虑）
- Twitter/X Ads 设置（账户基础、跟踪、受众设置、创意）
- TikTok Ads 设置（账户基础、像素和跟踪、受众设置、创意）
- 通用上线前清单

## Google Ads 设置

### 账户基础

- [ ] 已创建并验证 Google Ads 账户
- [ ] 已添加账单信息
- [ ] 时区和货币设置正确
- [ ] 已向团队成员授予账户访问权限

### 转化跟踪

- [ ] 已在所有页面上安装 Google 标签
- [ ] 已创建转化操作（购买、潜在客户、注册）
- [ ] 已分配转化价值（如果适用）
- [ ] 已启用 Enhanced conversions
- [ ] 测试转化正确触发
- [ ] 从 GA4 导入转化（可选）

### Analytics 集成

- [ ] 已链接 Google Analytics 4
- [ ] 已启用自动标记
- [ ] GA4 受众在 Google Ads 中可用
- [ ] 已设置跨域跟踪（如果有多个域名）

### 受众设置

- [ ] 已验证再营销标签
- [ ] 已创建网站访客受众：
  - 所有访客（180 天）
  - 关键页面访客（定价、演示、功能）
  - 转化者（用于排除）
- [ ] 已上传客户匹配列表
- [ ] 已启用相似受众

### 广告系列准备

- [ ] 已创建否定关键词列表：
  - 通用否定词（free、jobs、careers、reviews、complaints）
  - 竞争对手否定词（如果需要）
  - 不相关的行业术语
- [ ] 已设置位置定向（包含/排除）
- [ ] 已设置语言定向
- [ ] 已配置广告投放时段（如果是 B2B，则为工作时间）
- [ ] 已考虑设备出价调整

### 广告扩展

- [ ] 附加链接（4-6 个相关页面）
- [ ] 附加宣传信息（关键利益、优惠）
- [ ] 结构化摘要（功能、类型、服务）
- [ ] 电话分机（如果电话潜在客户有价值）
- [ ] 潜在客户表单扩展（如果使用）
- [ ] 价格扩展（如果适用）
- [ ] 图片扩展（在可用处）

### 品牌保护

- [ ] 正在运行品牌广告系列（保护品牌词）
- [ ] 已考虑竞争对手广告系列
- [ ] 非品牌广告系列的品牌词已加入否定列表

## Meta Ads 设置

### Business Manager 基础

- [ ] 已创建 Business Manager
- [ ] 已验证业务（如果投放某些广告类型）
- [ ] 已在 Business Manager 内创建广告账户
- [ ] 已添加付款方式
- [ ] 已配置具有适当角色的团队访问权限

### 像素和跟踪

- [ ] 已在所有页面上安装 Meta Pixel
- [ ] 已配置标准事件：
  - PageView（自动）
  - ViewContent（产品/功能页面）
  - Lead（表单提交）
  - Purchase（转化）
  - AddToCart（如果电子商务）
  - InitiateCheckout（如果电子商务）
- [ ] 已设置 Conversions API（CAPI）进行服务器端跟踪
- [ ] Event Match Quality 分数 > 6
- [ ] Events Manager 中的测试事件

### 域名和聚合事件

- [ ] 已在 Business Manager 中验证域名
- [ ] 已配置 Aggregated Event Measurement
- [ ] 按重要性顺序优先处理前 8 个事件
- [ ] 为 iOS 14+ 跟踪优先处理网络事件

### 受众设置

- [ ] 已创建自定义受众：
  - 网站访客（全部、30/60/90/180 天）
  - 关键页面访客
  - 视频观看者（25%、50%、75%、95%）
  - 主页/Instagram 互动者
  - 已上传客户列表
- [ ] 已创建相似受众（1%、1-3%）
- [ ] 常用定向的保存受众

### 商品目录（电子商务）

- [ ] 已连接商品目录
- [ ] 商品 Feed 正确更新
- [ ] 已启用目录销售广告系列
- [ ] 已配置动态产品广告

### 创意素材

- [ ] 图像尺寸正确：
  - Feed：1080x1080（1:1）
  - Stories/Reels：1080x1920（9:16）
  - Landscape：1200x628（1.91:1）
- [ ] 视频格式正确
- [ ] 广告文案变体已准备好
- [ ] 所有目标 URL 中都有 UTM 参数

### 合规性

- [ ] 已声明特殊广告类别（如果涉及住房、信贷、就业、政治）
- [ ] 落地页符合 Meta 政策
- [ ] 广告中没有禁止的内容

  - 已上传联系人列表

### Lead Gen Forms（如果使用）

- [ ] 已创建 Lead Gen Form 模板

- [ ] 已上传文档（PDF、PowerPoint）

## 通用上线前清单

启动任何广告系列之前：

- [ ] 已通过真实转化测试转化跟踪
- [ ] 落地页加载速度快（<3 秒）
- [ ] 落地页适合移动设备
- [ ] UTM 参数正常工作
- [ ] 预算设置正确（每日 vs. 总预算）
- [ ] 开始/结束日期正确
- [ ] 定向与预期受众匹配
- [ ] 广告创意已批准
- [ ] 已通知团队启动
- [ ] 报告仪表板已准备好
