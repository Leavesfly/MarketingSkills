# 目录列表 — 完整参考

按层级组织的规范目录列表。DR（Domain Rating）值是近似值且会随时间变化 — 在制定计划前请通过 Ahrefs 或 Moz 验证。

**列说明：**
- **DR** — Domain Rating（Ahrefs）。数值越高 = 传递的链接权重越多。
- **Dofollow** — 反向链接是否传递 SEO 价值。Nofollow 列表仍然对引荐流量和品牌信号很重要。
- **Cost** — 除非注明，否则均为免费。

## 第 6 层 — "最佳" 列表文章（编辑外联）

严格来说不是目录 — 这些是高 DR 域名上的博客文章，你可以通过冷外联被收录其中。通常比目录更有价值，因为它们结合了 dofollow 反向链接、编辑信任 + 市场中买家流量 + AI 引用权重。

**查找机会的搜索模式：**
- `"best [类别] tools" 2026`
- `"best [竞争对手] alternative"`
- `"top AI [类别]"`
- `"[类别] tools review"`

**外联模板（简短）：**
> Hey [name], saw your post on [best X tools]. We launched [product] recently — thought it might be worth a mention. Happy to give you a free account + credits for readers. Here's a 60s demo: [link]. No worries if not a fit.

**目标：** 30 天内获得 10 次收录。每次收录 = 来自 DR 40–70 的 dofollow 反向链接 + 引荐流量 + AI 引用燃料。

## 验证

任何提交上线后，验证反向链接是否存在且为 dofollow。你可以：

1. **手动：** 打开列表，右键点击你的产品链接，"检查" → 检查是否有 `rel="nofollow"` 或 `rel="ugc"`。如果不存在，则链接为 dofollow。
2. **curl：** `curl -sIL https://directory.com/your-listing | grep -i link`
3. **SEO 工具：** Ahrefs Site Explorer → Backlinks → 按此目录域名过滤。

**每季度重新验证。** 目录有时会毫无预警地将所有出站链接改为 nofollow — 如果 DR 停止移动，检查你最大的入站链接是否已悄悄翻转。
