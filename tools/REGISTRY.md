## 按类别分类

### Analytics

跟踪用户行为、衡量转化并分析营销效果。

| Tool | Best For | MCP Available |
|------|----------|:-------------:|
| **ga4** | Web analytics, Google ecosystem | ✓ |
| **mixpanel** | Product analytics, event tracking | - |
| **amplitude** | Product analytics, cohort analysis | - |
| **posthog** | Open-source analytics, session replay | - |
| **segment** | Customer data platform, routing | - |
| **adobe-analytics** | Enterprise analytics | - |
| **plausible** | Privacy-focused analytics | - |

**Agent 建议**：如果使用 Google 生态系统，从 GA4 开始。使用 Mixpanel 或 Amplitude 进行更深入的产品分析。Plausible 适用于注重隐私的网站。

### SEO

用于关键词研究、排名跟踪和网站审计的搜索引擎优化工具。

| Tool | Best For | Notes |
|------|----------|-------|
| **google-search-console** | Free, authoritative search data | Direct from Google |
| **semrush** | Competitive analysis, keyword research | Comprehensive |
| **ahrefs** | Backlink analysis, content research | Best for links |
| **dataforseo** | SERP tracking, backlinks, on-page audits | Comprehensive API |
| **keywords-everywhere** | Quick keyword research, traffic estimates | Credit-based |

**Agent 建议**：Google Search Console 必不可少（免费）。添加 Semrush 或 Ahrefs 进行竞争研究。DataForSEO 用于程序化 SERP 数据。Keywords Everywhere 用于快速关键词查询。

**Agent 建议**：Shopify 用于电子商务。Webflow 用于营销网站。WordPress 用于博客。对于无头 CMS：Sanity 适用于开发者灵活的内容，Contentful 适用于企业级多语言，Strapi 适用于自托管/预算敏感型。参见 [headless CMS guide](../skills/content-strategy/references/headless-cms.md) 了解选择标准。

## CLI 工具

零依赖、单文件 Node.js CLI，适用于没有自带 CLI 的工具。参见 [`clis/README.md`](clis/README.md) 了解安装说明和用法。

所有 CLI 遵循一致的模式：
- **无依赖** — 仅需 Node 18+，使用原生 `fetch`
- **JSON 输出** — 可通过管道传递给 `jq`、保存到文件或用于脚本
- **环境变量认证** — 设置 `{TOOL}_API_KEY` 即可使用
- **一致的命令** — `{tool} <resource> <action> [options]`

## 支持 MCP 的工具

这些工具提供 Model Context Protocol 服务器，支持直接的 agent 交互：

- **ga4** - Google Analytics 4 数据访问
- **stripe** - 支付和订阅管理
- **mailchimp** - 邮件活动管理
- **google-ads** - 广告活动管理
- **resend** - 事务性邮件发送
- **zapier** - 工作流自动化 + SDK 用于 8,000+ 应用集成
- **zoominfo** - B2B 联系人和意向数据
- **clay** - 数据丰富和外展自动化
- **supermetrics** - 跨平台营销数据
- **coupler** - 营销数据管道
- **outreach** - 销售互动序列
- **crossbeam** - 合作伙伴生态系统数据
- **introw** - 合作伙伴关系管理

要使用 MCP 工具，请确保在您的环境中配置了相应的 MCP 服务器。

### Composio 集成

[Composio](integrations/composio.md) 通过单个 MCP 服务器为 500+ 工具提供托管 OAuth 和预建连接器。它为没有原生 MCP 服务器的工具添加 MCP 访问，包括 HubSpot、Salesforce、Meta Ads、LinkedIn Ads、Google Sheets、Slack、Notion 等。

- **设置**：`npx @composio/mcp@latest setup`
- **快速入门**：参见 [tools/composio/README.md](composio/README.md)
- **营销工具映射**：参见 [tools/composio/marketing-tools.md](composio/marketing-tools.md)

当您需要对 OAuth 密集型工具的 MCP 访问时使用 Composio。当有原生 MCP 服务器（GA4、Stripe、Mailchimp 等）时优先使用——它们具有更深入的覆盖范围。
