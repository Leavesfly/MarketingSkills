# Nitrosend

AI 原生邮件平台，将事务性邮件和营销邮件整合在一个技术栈中，完全通过 MCP 由 AI 助手控制。无需传统仪表板——通过提示词构建序列、活动和自动化。

## 功能

| 集成方式 | 可用 | 说明 |
|-------------|-----------|-------|
| API | ✓ | REST API 可用 |
| MCP | ✓ | 完整 MCP 支持——主要集成方式 |
| CLI | - | 不可用 |
| SDK | - | 直接使用 MCP 或 API |

## 身份验证

- **类型**：API Key
- **MCP 设置**：将 Nitrosend MCP 服务器添加到 Claude Code / AI 助手配置中
- **自带基础设施**：可选择使用自己的 SendGrid、Postmark、SES 或 Resend 密钥
- **获取访问权限**：在 nitrosend.com 注册——免费套餐初始包含 8K 封邮件，之后每月 500 封

## 定价

| 套餐 | 费用 | 数量 |
|------|------|--------|
| Free | $0 | 初始 8K，之后每月 500 封 |
| Hobby | $20/月 | 每月 25,000 封 |
| Pro | $100/月 | 每月 150,000 封 |
| Scale | $300/月 | 每月 500,000 封 |

所有套餐联系人数量无限制——按发送的邮件数量付费，而非按订阅者数量。

## 独特之处

- **AI 优先**：专为 Claude、ChatGPT、Codex、Cursor、Gemini、Windsurf 控制而设计——而非人类点击仪表板操作
- **统一的事务性 + 营销邮件**：单一平台同时支持两者，使用独立的基础设施
- **自动优化**：根据参与度持续测试主题行、发送时间和内容
- **自动配置的送达率**：DKIM、SPF、DMARC 和专用 IP 预热自动处理
- **迁移友好**：可从 Mailchimp、Klaviyo、ActiveCampaign、HubSpot 导入

## 常见 Agent 操作（通过 MCP）

### 创建邮件序列

```
"为新 SaaS 试用用户创建一个 5 封邮件的入门序列。
第 1 封：欢迎 + 首要操作（立即发送）
第 2 封：核心功能亮点（第 2 天）
第 3 封：用例/成功案例（第 4 天）
第 4 封：跟进 + 提供支持（第 7 天）
第 5 封：升级提示（第 12 天）"
```

Nitrosend 构建序列、安排时间并发送——无需在仪表板中手动设置。

### 发送事务性邮件

```
"向 user@example.com 发送密码重置邮件，重置链接有效期为 1 小时。"
```

### 创建活动

```
"为 90 天未打开邮件的订阅者创建一个重新参与活动。
主题行变体：[A] '我们想念你'，[B] '仍然对 [主题] 感兴趣吗？'
测试两者，4 小时后向剩余列表发送获胜版本。"
```

### 检查序列表现

```
"显示入门序列的打开率、点击率和退订数。"
```

### 导入列表

```
"从我们的 Mailchimp 导出文件中导入这个包含 2,000 名订阅者的 CSV。"
```

## 送达率设置

Nitrosend 在注册时自动处理：
- DKIM 签名
- SPF 记录配置
- DMARC 策略
- 专用 IP 配置（Pro+）
- IP 预热计划

对于 BYO 套餐用户：使用自己的 SendGrid、Postmark、SES 或 Resend 账户，Nitrosend 通过你的基础设施路由。

## 使用场景

- 通过 AI 构建邮件序列，无需接触仪表板
- 已将 Claude Code 或其他 AI 编码工具作为主要工作流的团队
- 将事务性邮件（密码重置、收据）和营销邮件（培育、活动）整合在一处
- 快速序列原型设计——描述序列，即可构建
- 从 Mailchimp/Klaviyo 迁移并希望未来使用 AI 控制

## 何时选择其他方案

- **Customer.io** — 如果需要复杂的事件驱动分支逻辑和行为触发器
- **Klaviyo** — 如果在电商领域且需要深度 Shopify 集成
- **Resend** — 如果只需要事务性邮件且偏好纯 API/代码方式
- **Kit** — 如果是创作者或以通讯为主
