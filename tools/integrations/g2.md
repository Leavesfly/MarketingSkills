# G2

面向 B2B 买家的软件评论和研究平台。访问评论、产品数据、竞争对手比较和买家意图信号。

## 功能

| 集成方式 | 可用 | 说明 |
|-------------|-----------|-------|
| API | ✓ | Reviews, Products, Reports, Categories, Tracking |
| MCP | - | 不可用 |
| CLI | ✓ | [g2.js](../clis/g2.js) |
| SDK | - | REST API，采用 JSON:API 格式 |

## 身份验证

- **类型**：API Token
- **Header**：`Authorization: Token token={YOUR_API_TOKEN}`
- **Content-Type**：`application/vnd.api+json`（JSON:API）
- **获取令牌**：G2 Admin Portal > Integrations > API Tokens
- **文档**：https://data.g2.com/api/docs

## 常见智能体操作

### 列出评论（调查回复）

```bash
GET https://data.g2.com/api/v1/survey-responses?page[size]=25&page[number]=1

Headers:
  Authorization: Token token={API_TOKEN}
  Content-Type: application/vnd.api+json
```

### 获取特定评论

```bash
GET https://data.g2.com/api/v1/survey-responses/{id}

Headers:
  Authorization: Token token={API_TOKEN}
  Content-Type: application/vnd.api+json
```

### 按产品过滤评论

```bash
GET https://data.g2.com/api/v1/survey-responses?filter[product_id]={product_id}&page[size]=25

Headers:
  Authorization: Token token={API_TOKEN}
  Content-Type: application/vnd.api+json
```

## API 模式

G2 遵循 JSON:API 规范（https://jsonapi.org/）：
- 响应使用 `data`、`attributes`、`relationships`、`meta` 结构
- 分页：`page[number]` 和 `page[size]` 查询参数
- 过滤：`filter[field]=value` 查询参数
- 评论默认按最新优先返回（每页默认 10 条）

## 关键指标

### 评论指标
- `star_rating` - 总体星级评分
- `title` - 评论标题
- `comment_answers` - 结构化评论回复（喜欢、不喜欢、推荐）
- `submitted_at` - 评论提交日期
- `is_public` - 评论是否公开可见

### 产品指标
- `name` - 产品名称
- `slug` - G2 上的 URL slug
- `avg_rating` - 平均星级评分
- `total_reviews` - 评论总数
- `category` - G2 类别 placement

### 买家意图（跟踪）
- `company_name` - 访问公司名称
- `page_visited` - 访问的 G2 页面 URL
- `visited_at` - 访问时间戳
- `activity_type` - 买家活动类型

## 参数

### 分页
- `page[number]` - 页码（默认：1）
- `page[size]` - 每页项目数（默认：10，最大：100）

### 评论过滤器
- `filter[product_id]` - 按产品 ID 过滤
- `filter[state]` - 按评论状态过滤

### 跟踪过滤器
- `filter[start_date]` - 开始日期（YYYY-MM-DD）
- `filter[end_date]` - 结束日期（YYYY-MM-DD）

## 何时使用

- 监控和分析软件产品评论
- 跟踪来自 G2 访问者的买家意图信号
- 提取竞争对手比较数据用于定位
- 将评论数据导入 CRM 或营销自动化系统
- 基于 G2 评论构建社会证明内容
- 跟踪 G2 类别排名和报告 placement

## 速率限制

- 每个 API 令牌每小时 10,000 次请求
- 对 429 响应实施指数退避
- 尽可能缓存结果以减少 API 调用
